## 🖐🏻 项目介绍

本项目的目标是基于 `Gatsby` 和 `Strapi` 搭建一个完整的博客系统。样式上则以 `Ghost` 博客主题 ` Joemla` 为蓝本，计划复刻 `Ghost` 博客  90% 以上的样式和 80% 以上的功能。

本项目涉及以下技术栈：

* GraphQL
* Emotion
* React
